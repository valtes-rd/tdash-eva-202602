# 第 4 回 T-DASH\_問題

## 出題範囲

T-DASH エバンジェリスト認定資格試験では、以下が出題範囲となります：

1. テスト観点をもとに WEB アプリケーションの自動化ができること
   ※ 検証していない自動化は採点されません
2. キャプチャーツール／キャプチャー＆リプレイで適切な XPath を取得できること
3. キャプチャーツールで取得できない要素（表示位置が変動する要素など）を DevTools 等で取得し、一意に特定できること
4. 切り替え作業（iframe、新しいウィンドウ）を適切に扱えること
5. Loading 表示などがある不安定サイトでも安定した自動化ができること
6. 共通ステップをライブラリ化（動作セット）して保守性を高められること
7. データドリブンで複数データパターンを効率的に自動化できること
